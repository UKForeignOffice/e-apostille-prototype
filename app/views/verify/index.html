Before you start

If you received a paper version, you'll need:

  • the date the Apostille was issued
  • the Apostille number

If you received an electronic PDF file (legalised document with an e-Apostille), you'll need:

  • the date the Apostille was issued
  • the 16 character verification code
All the details above can be found on the Apostille certificate.
